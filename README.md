# RandomImageViewer
<div>
Приложение для просмотра рандомных картинок с котами из интернета с возможностью сохранения понравившихся.
  Проект построен по шаблону <b>MVVM</b>. <br>
  <br> В проекте использованы: 
  <ul>
    <li>Kotlin</li>
    <li>Jetpack Compose</li>
    <li>Retrofit</li>
    <li>Room</li>
    <li>Coil</li>
    <li>Coroutines</li>
  </ul>
